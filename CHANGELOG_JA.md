## [1.0.0] - 2026-05-30

初回パブリックリリース。

### 新機能

#### 動画入力
- 動画ファイル選択（MKV、MP4、MOV、AVI 対応）
- 動画メタ情報表示（解像度、長さ、フォーマット、ファイルサイズ）
- サムネイルプレビュー
- カスタムプロトコル `mojioko-media://` によるアプリ内動画再生（HTTP Range 対応）

#### 音声トラック選択
- マルチトラック音声の列挙（コーデック、チャンネル、サンプリングレート表示）
- トラック選択 UI、デフォルトトラック自動選択
- OBS マルチトラック録画向けに最適化

#### Whisper モデル管理
- 3 種類のモデル対応：`small`、`medium`、`large-v3`（Systran faster-whisper int8 量子化版）
- HuggingFace からのモデルダウンロード（ファイル単位の進捗表示）
- ダウンロード前のディスク容量チェック
- キャンセル可能なダウンロード（部分ファイルの自動クリーンアップ）
- アクティブモデルの切替、アンインストール、フォルダアクセス

#### 文字起こし
- faster-whisper Python sidecar（Python ランタイム不要、PyInstaller でバンドル）
- CPU + int8 量子化（GPU 不要、Windows PC で動作）
- VAD（音声検出）：閾値・継続時間の調整可能
- ビームサイズ調整（1-20）
- 11 言語対応（自動検出、日本語、英語、中国語、韓国語、スペイン語、フランス語、ドイツ語、ポルトガル語、ロシア語、アラビア語）
- リアルタイム進捗（1% 単位）
- 確認ダイアログ付きキャンセル機能
- 動画幅に合わせた自動改行（オプション）

#### 字幕エディタ（Step 2）
- テーブルビュー：# / 時間 / サイズ / スタイル / テキスト / 状態 / 操作 列
- 5 タブフィルタ：すべて / 出力対象 / 編集済み / 警告あり / 削除済み
- インラインテキスト編集（Ctrl+Enter / blur でコミット）
- インライン時間編集（HH:MM:SS.cc 形式）
- 時間調整ダイアログ：±1 秒 / ±0.1 秒 ステッパー、長押しリピート、前後行スナップ
- 行追加（開始時刻に応じた自動挿入）
- ソフト削除と復元
- 元の状態にリセット（テキスト / 時間 / スタイル）
- Undo / Redo（最大 100 ステップ）
- 時間編集後の自動ソート
- フォーカス行への自動スクロール（framer-motion アニメ完了後）
- テキスト書き出し（`.txt`、`\N` 改行除去）
- SRT 書き出し（UTF-8 BOM、DaVinci Resolve 互換）
- 動画プレビューパネル：再生・シーク、再生時刻と字幕の自動連動

#### 字幕スタイル設定
- フォントサイズ：30-200 px
- 文字色（HEX カラーピッカー、セッション履歴付き）
- アウトライン色（同上）
- アウトライン太さ：0-10 px
- フェード on/off
- フェード時間：0.1-0.5 秒（アプリ全体の設定）

#### 警告・出力可否判定
- **出力対象** = `!isDeleted && !emptyText`
- **焼き込み対象** = `isOutputTarget && !timeInvalid && !overDuration && !invalidSize`
- ステータスバッジ表示：時間不正、時間超過、時間重複、サイズ不正、はみ出し、テキスト未入力、削除済み
- はみ出し箇所の赤色強調表示

#### 動画焼き込み（Step 3）
- 字幕位置：水平（左 / 中央 / 右）× 垂直（上 / 下）
- 縦マージン調整（px）
- 字幕背景：on/off、色（黒 / 白）、不透明度（0-100%）
- **出力形式選択（MP4 / 入力と同じ形式）**
  - デフォルト：**MP4**（YouTube Shorts、TikTok、Instagram Reels 向け最適化）
  - MP4 モードでは `-f mp4 -movflags +faststart` を追加（ストリーミング配信用）
- エンコーダ自動検出：h264_nvenc → h264_amf → h264_qsv → h264_mf（フォールバック）
- エンコーダ明示指定 + フォールバック警告
- 音声出力モード：simple（AAC 192kbps ミックスダウン）または preserve（`-c:a copy`）
- 自動生成ファイル名：`{stem}_subtitled_{timestamp}.{ext}`
- ネイティブ保存ダイアログ（デフォルト `~/Videos`）
- 上書き確認ダイアログ
- ASS 字幕生成（libass 互換タグ）
- リアルタイム進捗バー（0.1% 単位）
- 部分ファイル削除付きキャンセル機能
- 完了後アクション：ファイルを開く、フォルダで表示、再書き出し、寄付

#### 多言語 UI
- 2 言語対応：日本語（デフォルト）/ 英語
- ネイティブメニューバーの翻訳
- ロケール別 JSON 名前空間
- 設定の永続化

#### ネイティブメニューバー
- ファイル → 終了（Ctrl+Q）
- ツール → 設定（Ctrl+,）
- ヘルプ → 使い方ガイド
- ヘルプ → OBS 設定ガイド
- ヘルプ → プロジェクトを支援する（寄付ダイアログ）
- ヘルプ → ログフォルダを開く
- ヘルプ → ダウンロードページを開く
- ヘルプ → このアプリについて

#### コマンドパレット（Ctrl+K）
- 5 カテゴリ：ナビゲーション / ファイル / 編集 / 設定 / ヘルプ
- 全主要アクションへのクイックアクセス

#### キーボードショートカット
- グローバル：Ctrl+K、Ctrl+,、Ctrl+/、Ctrl+O、Ctrl+Q
- Step 2：Ctrl+Z、Ctrl+Y、Ctrl+S、Ctrl+N、Delete、Ctrl+R

#### 設定ダイアログ
- 表示言語（日本語 / 英語）
- フェード時間（0.1-0.5 秒）
- `%APPDATA%\MOJIOKO\settings.json` への永続化
- 破損設定の自動復旧
- Step 3 設定の自動リセット（Step 1 への遷移時：burnin、subtitleBackground、audioMode、outputContainer）

#### 寄付ダイアログ
- 3 つの寄付チャネル：Buy Me a Coffee → BOOTH → GitHub Sponsors
- ヘルプメニューと Step 3 完了画面からアクセス可能

#### エラーハンドリング・ログ
- 型付きエラークラス
- トースト通知（成功 / 警告 / エラー / 情報）
- ログファイル：`%APPDATA%\MOJIOKO\logs\mojioko.log`（5MB ローテーション、3 世代保持）
- 起動時環境ダンプ（OS、Electron、Chromium、Node、GPU、ffmpeg エンコーダ）
- 未捕捉例外のハンドリング

#### セキュリティ・プライバシー
- 起動時の外部通信なし
- テレメトリ・解析なし
- サンドボックス化されたレンダラープロセス（contextIsolation + sandboxed preload）
- 厳格なコンテンツセキュリティポリシー
- 外部 URL の allowlist
- 書き込みパスを `~/` 配下に制限
- `mojioko-media://` allowlist
- Whisper モデル ID 検証

#### インストーラ（NSIS）
- マルチステップウィザード（ようこそ → ライセンス → インストール先 → 確認 → 進捗 → 完了）
- EULA 同意画面（日本語 / 英語、インストーラ言語に応じて自動切替）
- ユーザー単位インストール（管理者権限不要）
- カスタムインストール先対応
- デスクトップ + スタートメニューショートカット
- 「インストール後に起動」チェックボックス
- アンインストーラ（ユーザー設定保持）
- 同梱：PyInstaller ビルドの Python sidecar、LGPL ffmpeg shared build、Noto Sans JP フォント、サードパーティライセンス

### 既知の制限事項

- **コード署名なし**：初回起動時に SmartScreen 警告が表示される場合があります
- **自動更新なし**：ヘルプ → ダウンロードページを開く から手動で確認
- **macOS / Linux**：v1.0.0 では未対応（Windows のみ）
- **同梱フォント 1 種類のみ**：Noto Sans JP SemiBold（フォント切り替え機能は今後実装予定）
- **文字色・アウトライン色の不透明度なし**：100% 不透明のみ（透過値設定は今後実装予定）
