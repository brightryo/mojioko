📝 Feedback Form

We collect feedback via Google Forms.

Send Feedback

📋 Tips Before Sending

Q1 What to include in bug reports
  • MOJIOKO version (shown at top left of the app)
  • OS (Windows 10 / 11, etc.)
  • PC specs (CPU, memory, GPU)
  • Steps to reproduce
  • Log file
  • Screenshots

These details help us identify and fix issues quickly.

Q2 How to obtain the log file
  1. MOJIOKO menu: "Help" → "Open Log Folder"
  2. Select the latest log file (main.log, etc.) from the folder
  3. Attach it to the "Log File" field in Google Forms
Q3 How to capture screenshots

For Windows 11 / 10

  • Win + Shift + S: Snipping selection (recommended)
  • PrintScreen: Full screen capture

After capturing, attach the image to the "Screenshots" field in Google Forms.

Q4 About the information you submit
  • The information submitted through this form will be used solely to improve MOJIOKO
  • If your submission contains personally identifiable information, please consider removing it before sending
  • Please check that screenshots do not show personal information
  • Log files may contain your username (PC login name)
  • If you provide an email address, we may use it to confirm details or ask for additional information

About disclosure

We may use feedback to discuss publicly on GitHub Issues. Personal information will be removed when shared publicly.
