User Guide

FAQ, OBS Setup, and More

Q1 Which Whisper model should I download and use?

You can choose from 3 models in the in-app "Whisper Model" screen. We recommend medium.

medium ★ Recommended (Balanced)

  • File size: ~1.5 GB
  • Memory usage: Medium
  • Processing speed: Medium
  • Accuracy: High

Start with this model.

small Lightweight & Fast

  • File size: ~466 MB
  • Memory usage: Low
  • Processing speed: Fast
  • Accuracy: Standard

Use as alternative if medium feels heavy or slow.

large-v3 Highest Accuracy

  • File size: ~3 GB
  • Memory usage: High (16 GB+ RAM recommended)
  • Processing speed: Slow
  • Accuracy: Best

Upgrade if medium's accuracy is not sufficient.

How to Choose

  1. Start with medium
  2. If accuracy is not enough → Upgrade to large-v3
  3. If processing is slow → Downgrade to small

Notes

  • Downloaded models are saved and ready to use next time
  • You can switch between multiple models
  • You can download models anytime from the "Whisper Model" screen
  • Keep the app open during download
Q2 I want to transcribe only microphone audio

When game BGM or sound effects are saved on the same track as microphone audio, it becomes difficult to identify the transcription target and accuracy drops.

To solve this, record microphone audio on a separate track. (The video file's internal structure changes, but it's still a single output file.)

By assigning microphone audio to Audio Track 2, you can target only Track 2 for transcription, improving accuracy.

OBS Audio Track Configuration

■ Settings Screen
  1. Open OBS "Settings"
  2. Select "Output" from the left menu
  3. Change "Output Mode" to "Advanced"
  4. Select the "Recording" tab
  5. Check "Audio Track" "2"
  6. Click "Apply" → "OK"
■ Audio Mixer Configuration
  1. In the main window, click the "Options" button in the Audio Mixer
  2. "Advanced Audio Properties" dialog opens
  3. Check "Tracks" "2" for the microphone row
  4. Click "Close"

MOJIOKO Configuration

Select the recorded video file in MOJIOKO, choose "Track 2" as the transcription target audio track, and run transcription. You can now transcribe only the microphone audio.

Notes

Transcription accuracy can be fine-tuned via "Advanced Settings" at the bottom of MOJIOKO's transcription screen:

  • VAD (Voice Activity Detection) threshold
  • Beam size
  • Language specification (auto-detect / specific language)
Q3 "Windows protected your PC" appears during installation

This is a warning from Microsoft's SmartScreen feature. MOJIOKO is currently not code-signed, so this warning appears on first installation.

How to Proceed

  1. Click "More info"
  2. Click the "Run anyway" button that appears
  3. The installer will launch
