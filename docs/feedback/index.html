不具合報告 & ご要望

MOJIOKO の改善のため、ご意見・ご要望をお聞かせください

📝 フィードバックフォーム

Google Forms にて受け付けています。
GitHub アカウントは不要で、スクリーンショットやログファイルも添付できます。

フィードバックを送る

📋 送信前のヒント

Q1 不具合報告に含めて欲しい情報
  • MOJIOKO のバージョン（アプリ左上に表示）
  • OS（Windows 10 / 11 等）
  • PC のスペック（CPU、メモリ、GPU）
  • 再現手順
  • ログファイル
  • スクリーンショット

これらの情報があると、原因究明と修正対応がスムーズになります。

Q2 ログファイルの取得方法
  1. MOJIOKO のメニュー：「ヘルプ」→「ログフォルダを開く」
  2. 表示されたフォルダの最新ログファイル（main.log 等）を選択
  3. Google Forms の「ログファイル」項目に添付
Q3 スクリーンショットの撮り方

Windows 11 / 10 共通

  • Win + Shift + S：範囲選択スクリーンショット（推奨）
  • PrintScreen：全画面スクリーンショット

撮影後、画像を Google Forms の「スクリーンショット」項目に添付してください。

Q4 送信される情報について
  • このフォームで送信された情報は、MOJIOKO の改善目的にのみ使用します
  • 個人を特定できる情報を含む場合は、送信前に削除をご検討ください
  • スクリーンショットに個人情報が映り込んでいないかご確認ください
  • ログファイルにはユーザー名（PC のログイン名）が含まれる場合があります
  • メールアドレスを入力した場合、確認や追加情報のお伺いに使用することがあります

公開について

頂いたフィードバックを参考に、GitHub Issues 等で公開して議論する場合があります。公開時は個人を特定できる情報を取り除きます。

❓ 使い方の質問の場合

使い方の質問は、まず使い方ガイドをご確認ください。
よくある質問への回答が掲載されています。
