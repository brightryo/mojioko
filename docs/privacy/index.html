プライバシーポリシー

MOJIOKO におけるユーザーデータの取り扱いについて

最終更新日: 2026 年 6 月 27 日 ／ バージョン 1.0

1. 基本方針 ― すべての処理はローカル PC で完結します

MOJIOKO（以下「本アプリ」）は、動画ファイルからの文字起こし、字幕の編集、字幕の焼き込みといったすべての処理をユーザーの PC 内で完結させるデスクトップアプリケーションです。

本アプリは、ユーザーの動画ファイル、音声、生成された字幕、編集内容、その他の個人データを外部のサーバーに送信・アップロード・収集しません。

本アプリは、利用状況の追跡・解析（テレメトリ／アナリティクス／クラッシュレポート）を一切行いません。広告 SDK、トラッキング SDK、第三者の解析ツールは組み込まれていません。

2. アプリが外部と通信するケース

本アプリは、起動時や利用中に外部サーバーへ自動的に通信することはありません。ユーザーの明示的な操作によって、以下に限り外部通信が発生します。

2.1 Whisper モデルのダウンロード（Hugging Face）

文字起こしには OpenAI Whisper モデル（large-v3、large-v3-turbo など、約 1.5〜3 GB）が必要です。これらのモデルは、ユーザーが本アプリ内で「モデルをダウンロード」を選択したときに、Hugging Face（huggingface.co）からダウンロードされます。

ダウンロードされるのは公開されているモデルファイルのみです。ユーザーの動画・音声・個人データが Hugging Face に送信されることはありません。

Hugging Face との通信には、Hugging Face 社のプライバシーポリシー（https://huggingface.co/privacy）が適用されます。

2.2 追加フォントのダウンロード（GitHub Releases）

字幕に使用する追加フォント（Dela Gothic One、Reggae、Yusei Magic など、いずれも SIL Open Font License のフォント）は、ユーザーが本アプリ内で「フォントをダウンロード」を選択したときに、本アプリのGitHub Releases ページからダウンロードされます。

ダウンロードされるのはフォントファイルのみで、ユーザーデータは送信されません。GitHub との通信には GitHub のプライバシーポリシーが適用されます。

2.3 配布ページ・ヘルプリンクの外部ブラウザ起動

ヘルプメニューの「ダウンロードページを開く」「使い方ガイドを開く」「フィードバックフォームを開く」「GitHub Sponsors / BOOTH / Buy Me a Coffee」などの項目は、外部ブラウザで該当 URL を開くだけです。本アプリ内から HTTP リクエストを行うわけではなく、その後の通信はブラウザと各サイトの間で完結します。

2.4 自動アップデート確認は行いません

本アプリは electron-updater 等の自動アップデート機構を組み込んでいません。起動時や定期的にバージョン情報を外部サーバーへ問い合わせることはありません。

新しいバージョンの公開状況は、ヘルプメニューから配布ページ（GitHub Pages）をブラウザで開いて確認してください。

3. フィードバックフォーム（Google Forms）について

本アプリには、不具合報告・ご要望を受け付けるためのフィードバックフォームがあります。フォームの実体は Google Forms で提供されており、ヘルプメニューまたは配布ページから外部ブラウザで開かれます。

フォームを通じて開発者に届くのは、ユーザーが自発的にフォームに入力・添付した情報のみです。本アプリがバックグラウンドで自動送信することはありません。

メールアドレスを含む個人を特定する情報の入力は必須としていません。ログファイルやスクリーンショットを添付する場合は、ユーザー名（PC のログイン名）や個人情報が映り込んでいないかをご確認のうえ送信してください。

Google Forms の利用には、Google のプライバシーポリシー（https://policies.google.com/privacy）が適用されます。

4. 寄付サービスについて

本アプリのヘルプメニューには、開発支援のための寄付サービス（Buy Me a Coffee、BOOTH、GitHub Sponsors）への外部リンクが含まれます。これらは外部ブラウザでの遷移先サービスにおいて決済が行われるものであり、本アプリはユーザーの決済情報・カード情報・住所等に一切関与・取得しません。

5. 関与する第三者サービスの一覧

  • Hugging Face ― Whisper モデルファイルの配布元（ユーザーが「モデルをダウンロード」を選択したときのみ通信）
  • GitHub ― 本アプリ本体および追加フォントの配布、Issues トラッカー、GitHub Sponsors（ユーザーが該当機能を選択したときのみ通信）
  • Google（Google Forms） ― フィードバックフォームの提供（ユーザーが自発的にフォームへ入力・送信したときのみ）
  • Buy Me a Coffee、BOOTH ― 寄付の決済サービス（ユーザーが外部リンクを開いた場合のみ）

いずれも本アプリ内で完結する処理には関与せず、ユーザーの明示的な操作によって初めて通信が発生します。

6. ローカルに保存されるデータ

本アプリは、以下のデータをユーザーの PC のローカルストレージにのみ保存します。これらは外部に送信されません。

  • アプリ設定（テーマ、言語、字幕スタイル既定値など）― %APPDATA%/MOJIOKO/ 配下
  • ダウンロード済み Whisper モデルファイル ― %APPDATA%/MOJIOKO/ 配下
  • ダウンロード済み追加フォントファイル ― %APPDATA%/MOJIOKO/ 配下
  • ログファイル（不具合調査用、ユーザーが任意で添付）― %APPDATA%/MOJIOKO/logs/ 配下
  • ユーザーが書き出した字幕ファイル・動画ファイル ― ユーザーが指定した任意の場所

本アプリをアンインストールする場合、ユーザー設定・モデル・フォント・ログのフォルダ（%APPDATA%/MOJIOKO/）は手動で削除してください。

7. 子どもからの情報収集について

本アプリは、いかなる年齢層のユーザーからも個人情報を収集しません。これは子ども（13 歳未満を含む）についても同様です。

8. お問い合わせ

本ポリシーに関するご質問、本アプリのプライバシーに関するご懸念は、以下のいずれかの方法でお寄せください。

  • GitHub Issues ― https://github.com/brightryo/mojioko/issues（GitHub アカウントが必要）
  • フィードバックフォーム ― 配布ページ内「不具合報告 & ご要望」より（GitHub アカウント不要）

9. ポリシーの改定

本ポリシーは、本アプリの機能変更や法令の変更等に応じて改定されることがあります。改定があった場合は、本ページの「最終更新日」と「バージョン」を更新して公開します。重要な変更がある場合は、配布ページ等で告知します。
