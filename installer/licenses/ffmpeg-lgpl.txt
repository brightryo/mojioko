4. Method of linking and how to relink
---------------------------------------

  MOJIOKO links to FFmpeg dynamically via shared libraries (.dll)
  shipped next to ffmpeg.exe inside MOJIOKO's installation
  directory.  A user who wishes to use a modified FFmpeg version
  can do so by replacing the shipped .dll / .exe files with
  binaries of the same FFmpeg major version (matching SOname /
  ABI), as permitted by LGPL §6(b) ("use a suitable shared
  library mechanism for linking with the Library").
