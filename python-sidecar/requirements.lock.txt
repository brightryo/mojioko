#
# The legacy python-sidecar/requirements.txt (a single `faster-whisper==1.2.1`
# line) is preserved for developers who create their own .venv via `pip install
# -r requirements.txt` and rely on pip to solve the transitive graph.  That
# path is NOT reproducible — this lock is.
#
# Certifi note: the bundled cacert.pem SHA-256 does not match any release we
# could bisect (2025.7 through 2026.6.17 all differ); pip's resolver picked
# 2026.6.17.  Certifi has no functional impact on MOJIOKO because the Python
# sidecar makes no HTTPS calls at runtime (models are pre-downloaded by the
# Node-side model-downloader, Silero VAD is bundled inside
# faster_whisper/assets/).  Documented for reproducibility only.
#
